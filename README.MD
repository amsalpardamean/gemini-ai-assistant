🚀 Gemini AI Integration API

Simple backend service untuk integrasi Google Gemini AI menggunakan Node.js + Express.
Project ini menyediakan endpoint untuk:

Generate text dari prompt
Transcribe / generate dari audio
Base structure siap production & mudah dikembangkan

📌 Features

✨ Text generation endpoint
🎤 Audio → Text / AI processing endpoint
📦 Simple Express REST API
🔐 Environment based configuration
🧩 Siap diintegrasikan ke Web / Chatbot

🏗️ Tech Stack
Tech	Purpose
Node.js	Runtime
Express.js	REST API server
Multer	File upload handling
Google Generative AI SDK	Gemini integration
dotenv	Environment config

⚙️ Installation
1️⃣ Clone project
git clone https://github.com/yourrepo/gemini-integration.git
cd gemini-integration

2️⃣ Install dependencies
npm install

3️⃣ Setup environment
Buat file .env
GEMINI_API_KEY=your_api_key_here

▶️ Run Server
node index.js

Server akan berjalan di:
http://localhost:3000
